Upload image to GitHub via GitHub API

This is a prototype to show how to upload and save an image to a GitHub repository directly from browser and without using any backend code. Demo is based on https://codepen.io/bitbug/pen/abjwmxz

You need to have write access to "Contents" of repository that you wish to upload to. You can generate a fine-grained personal access token on github at: https://github.com/settings/personal-access-tokens/new. Select repository that you wish to give permission to and then under repository permissions set access to "Contents" to read and write.
